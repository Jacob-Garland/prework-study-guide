HTML

  • HTML stands for Hyper Text Markup Language
  • HTML contains elements that make up the structure of all webpages.
  • Parent elements are elements with others inside, called child elements.
  • Declarations differ from elements, and determine things like the document type.
  • Declartions are NOT case-sensitive, but elements ARE.
  • Some elements are visible to users, some are not.
  • Elements are enclosed with angle brackets on either side of the tag.
  • Every element has an opening and closing tag. The closing tag will have a "/" after the left angle bracket.

HTML Cheatsheet

  • base: element that defines the base URL
  • link: element that connects to an external resource
  • meta: element that defines metadata, such as the character set or charset, description, keywords, author, and the viewport
  • div: creates a divisional line, or line break in the page
  • span: similar to "div", inline division for the page
  • h1, h2, h3, etc: headers for your webpage
  • p: paragraph bodies, blocks of texts, for organizational structure
  • ul, ol, li: Unordered Lists, Ordered Lists, and List Items. Used for organizing content.
  • img: used for an image element
  • br: line break
  • a: element that stands for anchor, creates links to the same or other webpages
  • href: a part of an element, no angle brackets. followed by = and URL for link in a relative path
  • rel: relationship between current document and the linked document resource

CSS

  • CSS stylesheets come in two formats: external and internal.
  • Internal stylesheets are contained in the head section of an HTML document, with the rules applying only to that page.
  • External stylesheets are kept in a file seperate from the HTML. Best for Seperation of Concerns.
  • The simpler the code, the better. When styling CSS, code multiple elements into the same style if you're going to be coding them the same anyways.
  • Some styles of CSS include coloring webages, fonts and their alignments, font sizes, font colors, design elements of images and backgrounds, and more.

    • style: an element that defines CSS
    • margin: property indicating amount of space around the outside of an element
    • padding: property indicating amount of space around content inside the element
    • border: property indicating to create a border
    • box-shadow: property that adds shadow effects around an elements frame
    • class attribute: a declaration for a new CSS rule based on the element name.

    Git

  • Git commands always start with Git. Bash commands will not. The following are commands you would enter in Gitbash as your Windows terminal
    • git status: checks what branch currently working within
    • git checkout -b branch-name: creates a new branch and switches to it
    • ls: lists directories
    • cd directory-name: changes dierctories
    • pwd: prints working directory to show file path in folders
    • cd ..: goes back one directory
    • mkdir directory-name: makes new directory
    • touch file-name.ext: creates a file as choosen extension
    • start file-name: opens a file
    • code .: opens VS to code in branch
    • ls -al ~/.ssh: lists all SSH keys
    • git revert: follow this command in the terminal with the first seven numbers of the SHA to revert to the previous version of code
    • git add -A: commits all files in working branch
    • git commit -m: follow this command with short descriptive text in quotes to commit
    • git checkout main: switches to main branch
    • git pull origin main: pulls feature branch into main branch for pull requests
    • git push origin: pushes changes to repository
